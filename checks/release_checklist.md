# Release Checklist

Use this final checklist before sending a PDF, publishing a page, or submitting a job-market packet.

- [ ] No `[TODO` markers remain in the file being shared.
- [ ] No placeholder brackets remain unless intentionally used in a template.
- [ ] All dates are current.
- [ ] All links open correctly.
- [ ] No confidential project, coauthor, review, or data details are included.
- [ ] PDF output has been reviewed visually.
- [ ] Filename includes version or date when useful.
- [ ] Application-specific materials name the correct institution and position.
