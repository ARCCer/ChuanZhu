# Job-Market Materials

This folder stores job-market-specific versions and drafts. These files should be adapted from the canonical facts in `source/`, but they may be framed differently for specific audiences.

Suggested contents:

- `cv_job_market.tex`: CV variant for applications.
- `research_statement_outline.md`: Research statement structure and evidence checklist.
- `teaching_statement_outline.md`: Teaching statement structure and evidence checklist.
- `cover_letter_template.md`: Reusable cover letter shell.
- `application_tracker.yml`: Optional tracker for roles, deadlines, and submitted materials.

Before sending applications, review both checklists in `checks/`.
